# CheckList de Conhecimento - PS 2026
**Eletrônica | Equipe Imperador**

## 1. Básico (tarefas obrigatórias)

* Escolha um dos sensores abaixo para estudar datasheet, escrever um código e ler em protoboard (Marcar um dia com o gerente no laboratório).
  * Hall 3144
  * ADXL345
  * Hall 49e
  * MAX6675
* Estudar os esquemáticos das ECUs;
* Ler o datasheet do MCP2515 e o STM32F411CEU6 e a partir disso apresentar suas principais características elétricas e funções;
* Leitura do regulamento da parte elétrica do carro;
* Estudar e aprender a usar o GitHub (necessário para entregas).

---

## 2. Complementar (para candidatos que desejam demonstrar proficiência excepcional)

* Assistir segundo módulo do curso de programação de STM32;
* Leitura do relatório de projeto de 2026 (parte de eletrônica);
* Desenvolver o esquemático de um circuito elétrico + PCB no Altium de um STM32F411CEU6 (se precisar de ajuda procure o gerente).

---

## 3. Entrega

* Você deverá realizar todas as atividades do tópico 1 para ter seu checklist de conhecimento validado.
* Relacionado ao estudo das ECUs, você deverá realizar uma apresentação sobre o que aprendeu para o subsistema (conversar com o gerente para marcar o dia).
* Todos os arquivos devem ser entregues através de um link público no SEU GitHub.
* Além disso, o tópico da leitura dos datasheets deve ser entregue em arquivos .md (mcp2515.md e stm32.md), da mesma forma, você deve entregar um breve resumo do regulamento eletrônico.

---
